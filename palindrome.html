🔁 Palindrome Program in Golang (Using ASCII Comparison)

🧠 Problem Statement

Write a program to check whether a given string is a palindrome using ASCII-based character comparison.

📚 What is ASCII?

ASCII (American Standard Code for Information Interchange) assigns a numeric value to each character. In Go, a string is a sequence of bytes (i.e., ASCII values for basic Latin letters).

By comparing characters using str[i] != str[len(str)-i-1], we are checking their ASCII values directly.

📄 Go Code

package main

import "fmt"

func main() {
    str1 := "madam1"
    fmt.Println("Given String is palindrome? ", isPalindrome(str1))
}

// isPalindrome checks if the string is a palindrome using ASCII value comparison
func isPalindrome(str string) bool {
    if str == "" {
        return false
    }

    for i := 0; i < len(str)/2; i++ {
        if str[i] != str[len(str)-i-1] {
            return false
        }
    }

    return true
}

✅ Output

Given String is palindrome?  false

Explanation: "madam1" is not a palindrome because the last character '1' (ASCII 49) does not match 'm' (ASCII 109).
